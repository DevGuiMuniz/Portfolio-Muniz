MURUMURU

Uma plataforma digital criada com o objetivo de promover profissionais da área da construção civil que trabalham de maneira autônoma. Desenvolvido em ReactJS, MySQL e utilizando uma API Java com o framework Spring Boot, o projeto visa facilitar o trabalho desses profissionais, proporcionando recursos que otimizam o tempo, diminuem o acúmulo de tarefas e promovem a visibilidade de suas habilidades e competências.
